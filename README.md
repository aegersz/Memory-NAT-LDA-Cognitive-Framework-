## Executive Summary

- Memory is not purely neuronal firing or synaptic change — it is also **fluidic**, governed by **lipid droplet differentials**.
- **NAT** acts as a memory-access substrate, interfacing with interoception, emotion, and time.
- Every **memory frame** is a composite of sensory, motor, virtual, and interoceptive inputs, all coordinated through NAT.
- The model explains not just how memory is stored, but how it is **ranked, accessed, imagined, and protected**.

## Core Components

### 1. Lipid Droplet Architecture (LDA)

The structural and metabolic state of lipid droplets within a cell.

- Not encoded by DNA or RNA — exists outside the genetic instruction set
- Governs:
  - Membrane fluidity
  - Signal weighting
  - Protein expression at the synapse
  - Transcriptional bias

### 2. Neural Adipose Tissue (NAT)

A macro-structure of lipid-rich tissue with fast access to neurites.

- Facilitates:
  - Omnidirectional thought propagation
  - Preconscious memory access
  - Emotional tagging
  - Biological clock synchronisation (via GABA/Glutamate oscillation)

### 3. Memory Frame Schema

Each memory is constructed around an **interneuron (IN)** hub and includes four node types:

| Node | Role |
|------|------|
| **Sensory Neuron (SN)** | Input — external perception |
| **Motor Neuron (MN)** | Action — behavioural output |
| **Virtual Frame (VF)** | Imagination / simulation / projection |
| **Interoceptive Node (INT)** | Internal state — mood, body, affect |

All nodes are NAT-modulated and neurotransmitter-weighted:

| Neurotransmitter | Role in this model |
|------------------|--------------------|
| Norepinephrine | Memory salience |
| Dopamine | Motivation (post-satiety signal) |
| Serotonin | Satiety / suppression |
| GABA / Glutamate | CNS clock / temporal routing |
| Acetylcholine | Attention gating |

## Fundamental Principles

### A. No Two Adjacent Cells Are Identical

Their difference lies in **LDA** — the fluidic, real-time readout of each cell's environmental history. This governs their functional role and memory access capacity, independent of genetic identity.

### B. Lipid Logic

A higher-order logic system operating outside the gene-protein loop, enabling:

- Non-coded, emergent intelligence
- Quasi-synaptic permission states and signal strength modulation
- Distributed, non-linear memory mapping

### C. Virtual Partner in Cognition

The VF node represents simulated memory projection — the cognitive internal "partner." Enabled by NAT dynamics and LD modulation. Underpins imagination, planning, and creative recombination.

### D. Interoception as Sixth Sense

NAT translates fluidic cellular state into mood, awareness, and internal perception. It filters memory recall and mediates reality acceptance.

### E. Time and Chronology

NAT maintains a temporal continuum. Memory is routed in one chronological direction via oscillatory balance of GABA and Glutamate. Sleep drives synaptic adaptation and is the primary consolidation window.

## Memory Updating and Persistence

- New memories are not written over old ones — they are **layered** into NAT-permitted networks.
- Persistent memories correlate with LDs that reinforce structural pathways.
- Imagination and creative recombination are simulations drawn from NAT-licensed frames.
- Some thoughts are understood as **virtual emulations** of sensory input — indistinguishable internally from perception.
